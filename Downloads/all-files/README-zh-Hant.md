# 多範本文件工具 · 使用說明

一個**純前端、零相依、完全離線**的網頁文件工具。開啟一個 HTML 檔案即可在瀏覽器內填寫並產生 Word 文件(.docx),無需安裝、無需連網、不上傳任何資料到伺服器。

---

## 核心特性

- **單檔執行**:雙擊用瀏覽器開啟即可使用,斷網也能正常產生文件。
- **零相依**:產生 Word 的邏輯全部以原生 JavaScript 手寫(手寫 zip 封裝 + docx XML),不引用任何 CDN 或外部函式庫。
- **隱私安全**:所有資料(包括上傳的證件圖片)僅在本機瀏覽器內處理,不連網、不上傳。
- **即時預覽**:左側填寫、右側 A4 版面即時預覽,所見即所得。
- **多範本**:內建四種範本,頂部下拉一鍵切換。
- **三語介面**:簡體中文、繁體中文、English,介面與產生文件的固定話術同步切換。

---

## 四種範本

### 範本1 · 在職(履歷)證明
完整的員工在職證明,欄位包括姓名、證件類型與號碼、崗位、入離職時間、部門、薪資、負責人資訊等,並可附上證件影印件頁。具有豐富的核驗規則(見下文)。

### 範本2 · 審批單申請
常見審批單,欄位包括申請人、部門、申請日期、審批事項類型(請假/報銷/採購/用章/其他)、事由說明、金額與幣別、起訖日期、審批人、備註。產生含資訊表、事由、簽字欄、蓋章位的審批單。

### 範本3 · 公告
標準公文公告版面,欄位包括發文單位、公告標題、公告編號、正文內容、落款單位、發佈日期。正文按空行自動分段、首行縮排,落款與日期右下對齊。

### 其他 · 自由編排
模組化文件編輯器,可自由增刪與排序模組:抬頭、標題、段落、資訊表、結論語、蓋章位、圖片頁。抬頭可上傳 logo、設公司名與標語;資訊表每行可單獨開關核驗。完全自由排版。

## 核驗規則(範本1)

以下核驗未通過時會禁止產生,並在對應區域給出提示:

- **姓名**:僅限純中文或純英文,不可混合;中文至少 2 字,英文至少 2 字母。
- **證件號碼**:按所選證件類型核驗位數。四類中國證件(居民身分證、港澳/台灣居民居住證、外國人永久居留身分證)額外核驗:18 位、第 7–14 位為合法出生日期(YYYYMMDD)、末位核驗碼正確。
- **崗位類型與年齡**:當證件為上述四類中國證件時,按出生日期(基準為系統當日)核驗周歲——全職≥18、實習≥16、兼職≥12;崗位為「其他」或證件為護照等不帶出生日期的證件,則不核驗。
- **入離職日期**:若選擇了離職日期,須晚於入職日期。
- **薪資金額**:若填寫,須為阿拉伯數字、首位非 0(允許 0 與小於 1 的小數)。
- **證件影印件**:可選,最多 4 張圖片;上傳則在文件末尾產生影印件頁。

## 輸入輔助

- **18 位中國證件**:輸入框內即時顯示為「六位地區碼 + 空格 + 八位生日 + 空格 + 後四位」(如 530112 20101213 6614),便於核對;核驗與文件輸出仍使用無空格的原始號碼。
- **日期選擇器**:入職、離職、申請、有效期等均為日期選擇器,統一顯示為 YYYY/MM/DD。
- **預設值**:負責人或上級、電郵、傳真留空顯示 N/A,負責人電話留空顯示預設號碼;有效期留空時自動取申請日期後 6 個月;申請日期留空用系統當天。
- **貨幣符號**:薪資幣別顯示為 CN￥ / HK$ / US$。

---

## 使用步驟

1. 用瀏覽器(Chrome / Edge / Safari 等)開啟 HTML 檔案。
2. 頂部選擇語言與範本。
3. 在左側表單填寫,右側即時預覽。
4. 核驗全部通過後,點擊右上角「下載 Word 文件」,產生的 .docx 用 Word 或 WPS 均可開啟。

---

## 說明與限制

- 網頁預覽按螢幕像素排版,Word 按 A4 紙張排版,兩者觀感一致但非逐像素相同。
- 瀏覽器原生日期選擇器外觀因瀏覽器而異,但輸出格式統一。
- 工具不含任何特定公司品牌,抬頭與單位名稱均由使用者自行填寫。
- 英文翻譯按 HR / 公文語境處理;若有公司慣用譯法可自行替換文件中的對應文字。
