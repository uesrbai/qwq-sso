# 多模板文档工具 · 使用说明

一个**纯前端、零依赖、完全离线**的网页文档工具。打开一个 HTML 文件即可在浏览器内填写并生成 Word 文档(.docx),无需安装、无需联网、不上传任何数据到服务器。

---

## 核心特性

- **单文件运行**:双击用浏览器打开即可使用,断网也能正常生成文档。
- **零依赖**:生成 Word 的逻辑全部用原生 JavaScript 手写(手写 zip 打包 + docx XML),不引用任何 CDN 或外部库。
- **隐私安全**:所有数据(包括上传的证件图片)仅在本机浏览器内处理,不联网、不上传。
- **实时预览**:左侧填写、右侧 A4 版式实时预览,所见即所得。
- **多模板**:内置四种模板,顶部下拉一键切换。
- **三语界面**:简体中文、繁体中文、English,界面与生成文档的固定话术同步切换。

---

## 四种模板

### 模板1 · 在职(履历)证明
完整的员工在职证明,字段包括姓名、证件类型与号码、岗位、入离职时间、部门、工资、负责人信息等,并可附上证件影印件页。带有丰富的校验规则(见下文)。

### 模板2 · 审批单申请
常见审批单,字段包括申请人、部门、申请日期、审批事项类型(请假/报销/采购/用章/其他)、事由说明、金额与币种、起止日期、审批人、备注。生成含信息表、事由、签字栏、盖章位的审批单。

### 模板3 · 公告
标准公文公告版式,字段包括发文单位、公告标题、公告编号、正文内容、落款单位、发布日期。正文按空行自动分段、首行缩进,落款与日期右下对齐。

### 其他 · 自由编排
模块化文档编辑器,可自由增删与排序模块:抬头、标题、段落、信息表、结论语、盖章位、图片页。抬头可上传 logo、设公司名与标语;信息表每行可单独开关校验。完全自由排版。

## 校验规则(模板1)

以下校验不通过时会禁止生成,并在对应区域给出提示:

- **姓名**:仅限纯中文或纯英文,不可混合;中文至少 2 字,英文至少 2 字母。
- **证件号码**:按所选证件类型校验位数。四类中国证件(居民身份证、港澳/台湾居民居住证、外国人永久居留身份证)额外校验:18 位、第 7–14 位为合法出生日期(YYYYMMDD)、末位校验码正确。
- **岗位类型与年龄**:当证件为上述四类中国证件时,按出生日期(基准为系统当日)核验周岁——全职≥18、实习≥16、兼职≥12;岗位为"其他"或证件为护照等不带出生日期的证件,则不核验。
- **入离职日期**:若选择了离职日期,须晚于入职日期。
- **工资数额**:若填写,须为阿拉伯数字、首位非 0(允许 0 与小于 1 的小数)。
- **证件影印件**:可选,最多 4 张图片;上传则在文档末尾生成影印件页。

---

## 输入辅助

- **18 位中国证件**:输入框内实时显示为"六位地区码 + 空格 + 八位生日 + 空格 + 后四位"(如 530112 20101213 6614),便于核对;校验与文档输出仍使用无空格的原始号码。
- **日期选择器**:入职、离职、申请、有效期等均为日期选择器,统一显示为 YYYY/MM/DD。
- **默认值**:负责人或上级、电邮、传真留空显示 N/A,负责人电话留空显示默认号码;有效期留空时自动取申请日期后 6 个月;申请日期留空用系统当天。
- **货币符号**:工资货币显示为 CN￥ / HK$ / US$。

---

## 使用步骤

1. 用浏览器(Chrome / Edge / Safari 等)打开 HTML 文件。
2. 顶部选择语言与模板。
3. 在左侧表单填写,右侧实时预览。
4. 校验全部通过后,点击右上角「下载 Word 文档」,生成的 .docx 用 Word 或 WPS 均可打开。

---

## 说明与限制

- 网页预览按屏幕像素排版,Word 按 A4 纸张排版,两者观感一致但非逐像素相同。
- 浏览器原生日期选择器外观因浏览器而异,但输出格式统一。
- 工具不含任何特定公司品牌,抬头与单位名称均由用户自行填写。
- 英文翻译按 HR / 公文语境处理;若有公司惯用译法可自行替换文档中的对应文字。
